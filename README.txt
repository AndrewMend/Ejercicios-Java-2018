Instrucciones para ejecutar los programas.

Todos los programas fueron desarrollados en el IDE Eclipse Oxygen para Windows.
Para ejecutarlos, es importante tener instalado el IDE antes mencionado, así como los requerimientos para trabajar, como son JDK 9 y JRE.

Pasos a seguir:

1. Abrir el IDE Eclipse.
2. Ir a la opción file>import y en las opciones que aparecen seleccionar "General>Existing Projects into Workspace".
3. En la ventana navegar "Import Projects> Select root directory > Browse..." y ahí localizar la carpeta de este repositorio (Ejercicios-Java-2018).
4. Seleccionar la carpeta y presionar "Finish".
5. Se importarán los proyectos. Es necesario desplegar cada proyecto en el explorador de paquetes, para ejecutar en Eclipse. 
6. Ir al paquete modelo.
7. La clase "modelo.Principal" de cada proyecto es la que debe ejecutarse.
8. Para ejecutar es necesario buscar el icono "Run" en eclipse, el cual se representa por una flecha blanca con fondo verde.
9. Siguiendo estos pasos, el programa se ejecutará.
